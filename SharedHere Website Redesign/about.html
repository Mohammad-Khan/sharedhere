What is SharedHere all about?

SharedHere is an Android App for GeoTagged content sharing. It stores and facilitates the upload and download activity of content tagged with specific information shared with respect to GPS Coordinates. The SharedHere app has a wide array of uses, from casual users wanting to embark on a virtual scavenger hunt, to a commercial business distributing coupons to patrons in close proximity to its doors.

Built for Android Devices

Built on Eclipse

Written in Java

Server-scripting with PHP

Utilizes Google Maps API

Manages data with mySQL
